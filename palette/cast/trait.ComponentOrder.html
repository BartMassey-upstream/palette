pub trait ComponentOrder<C, P> {
    fn pack(color: C) -> P;

    fn unpack(packed: P) -> C;
}
Expand description

Packs and unpacks color types with some component order.

As an example, RGBA channels may be ordered as ABGR, ARGB, BGRA, or RGBA.
